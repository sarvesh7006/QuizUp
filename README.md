# 🎯 QuizUp Lite

A frontend-only quiz web app inspired by QuizUp, featuring category-based trivia, timed questions, and a simulated multiplayer experience.

---

## 🚀 Features

* 📂 Category selection (Movies, Series, Games)
* ❓ Multiple-choice questions (MCQs)
* ⏱️ Timer-based gameplay
* 🧠 Dynamic scoring system
* 🤖 Simulated opponent (fake multiplayer)
* 📊 Progress tracking (question count)
* 🏆 Result screen (Win/Lose/Draw)
* 💾 High score saved using localStorage
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)

---

## ⚙️ How It Works

1. Select a quiz category
2. Answer timed questions (10 seconds each)
3. Score updates based on correct answers
4. Simulated opponent score updates dynamically
5. Final result displays the winner

---

## 🧠 Simulated Multiplayer Logic

Instead of real-time players, the app generates a fake opponent:

* Random opponent name
* Score increases after each question
* Creates a real multiplayer-like experience without backend
